# Enigma

Cipher machine used by NAZI Germany during World War II.

## Build and Run

### Prerequisites
- GCC compiler
- Make utility

### Build the Project
Run the following command to compile the project:
```sh
make build
```

### Run the Project
Execute the following command to build and run the project:
```sh
make run
```

### Clean the Build
To remove the compiled files, run:
```sh
make clean
```
